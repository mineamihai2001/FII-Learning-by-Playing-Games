# Learning-by-Playing-Games

Sa se realizeze o aplicatie Web menita a oferi suport pentru invatarea componentelor principale ale unui computer sau dispozitiv mobil. Se vor lua in consideratie scopul, categoria, regulile, provocarile si manierele de interactiune cu jocul respectiv â€“ a se studia si Outline of games. Sistemul va monitoriza progresul inregistrat de fiecare utilizator in procesul de invatare, pe baza mai multor niveluri de dificultate. Se va genera, de asemenea, un clasament al celor mai de succes utilizatori, disponibil si ca flux de date RSS. Inspiratie: <https://wiki2.org/en/Computer_hardware>. Bonus: realizarea unui mini-joc de testare a cunostintelor dobandite pe baza sistemului Web implementat.
