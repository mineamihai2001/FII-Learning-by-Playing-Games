1. Introduction

1.1 Purpose

The purpose of the document is to present the details of the web application Learning by Playing games. It will present the features of the application, the interface and the way the application must work. This document will be used for a better understanding of the application by the first time users.

1.2 Intended Audience and Reading Suggestions

This document is intended for both developers and users of the application. The developers will be able to understand how the app works. The app can be used by the following categories of people: People who are new to computers and know a few things about them or next to nothing / people who already know the basics and are eager to advance in this field / advanced people who want to play games related to their field

1.3 Product Scope

The goal of the application is to help people get a better understanding of the machine that they use on a dayly basis in order for them to be fell more in control when they use a computer. Each user will be able to start their learning process relative to their skills and advance in their own pace. The users can choose when they want to learn and what they want to learn, the main feature of the app being flexibiliy.

2. Structure

2.1 Home

The homepage

The homepage contains the navbar, a quick description of the webpage and 2 buttons that change colors on hover and send you to start the learning process (GET STARTED) and to read more about the idea of the site (LEARN MORE)

2.2 Login

login

The login section uses the same idea as all the other login menus. All the buttons use a color changing animation on hover that adds a little bit of style to a simple looking page. The option to connect with an account from Google, Facebook or Microsoft is here for an easier process of login-in.

2.3 Sign-up

sign-up

If you don't have an account, when prompted on the login screen you can press a button to the sign-up page where you will create an account after accepting the terms and conditions. The register button also uses the same animations on hover as on the login page.

2.4 Lesson

lesson

The lesson page is a quick preview of how someone is going to learn new information about pc components. The navbar is here, so you can go wherever you want, on the left side of the screen you can visualize your progress until completion and when you decide you have finished you can continue with the next lesson using a button with zoom-in on hover.

2.5 Admin Console

admin

AdminConsole is a page on which an admin can create lessons and quizzes after he logs in. He can choose from the left side sub-lessons and questions by dragging them to the designated section or by typing their IDs.
